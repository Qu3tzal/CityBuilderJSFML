README
======

Présentation du projet
----------------------
Ce projet a pour but de simuler l'évolution d'une ville et plus particulièrement l'implantation de ses bâtiments.


Structure
---------
- src contient l'ensemble du code source
- res contient l'ensemble des ressources
- lib contient la bibliothèque JSFML
